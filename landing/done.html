Setup Instructions

🍎 Mac Installation
⚠️ macOS may block apps downloaded from the internet. Use the Terminal script below to install the app correctly.
  1. Download the .dmg file above
  2. Open Terminal and paste this script:
hdiutil attach ~/Downloads/clipmatic.video_0.1.0_signed.dmg cp -R "/Volumes/clipmatic.video/clipmatic.video.app" /Applications/ xattr -cr /Applications/clipmatic.video.app hdiutil detach "/Volumes/clipmatic.video" open /Applications/clipmatic.video.app
  1. Press Enter — the app will install and open automatically
  2. Add your API keys and start generating videos!
🪟 Windows Installation
  1. Download the .msi file above
  2. Double-click to run the installer
  3. If Windows SmartScreen appears, click "More info" → "Run anyway"
  4. Open the app, add your API keys and start generating videos!
